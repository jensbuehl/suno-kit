# 🎤 SUNO-Mastertool V 1.1.1

A Chrome Extension for downloading and editing LRC files (lyrics) and other stuff like cover images, videos and audios from Suno.com songs for creating lyric videos.

## ✨ Features

- **Auto-Load**: Automatically extracts LRC content via API and bearer auth token from Suno.com song pages
- **Text Editing**: 
  - Remove punctuation (preserves timestamps)
  - Convert text to UPPERCASE or lowercase
  - Ability to do your own edits inside the textbox
- **Export Options**:
  - Download LRC file
  - Copy to clipboard
  - Download MP3 directly from Suno
  - Download cover art directly from Suno
- **Multi-language**: German and English user interface

## 📦 Installation

1. Clone repository or download as ZIP
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select the extension folder

## 🚀 Usage

1. Open a song on [Suno.com](https://suno.com) (`suno.com/song/...`)
2. Click the extension icon in Chrome toolbar
3. LRC data loads automatically
4. Select desired editing options
5. Export the result as file or copy to clipboard
6. Download everything else you need from the page

## 🛠️ Editing Options

- **Remove punctuation**: Removes all punctuation from lyrics (timestamps remain intact)
- **Text transformation**: Converts all text to uppercase or lowercase
- **Text editing**: Make your own edits inside the textbox
